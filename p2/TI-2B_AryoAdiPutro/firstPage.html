Di minggu pertama saya belajar tentang HTML, lebih tepatnya adalah tag-tag yang ada di HTML. Ada tag untuk heading, tag untuk paragraf, untuk memberi garis horizontal, memberi jarak dan lain-lain. Berikut adalah beberapa rincian tagnya :

NO Nama Tag Tag Keterangan
1 Heading <h1> Tag ini adalah untuk menjadikan judul teks yang ada dalam tag
2 Paragraf <p> Tag ini digunakan untuk membuat paragraf pada HTML
3 Garis Horizontal <hr> Tag ini digunakan untuk membuat garis horizontal di halaman website
4 Jarak <br> Tag ini digunakan untuk membuat satu baris jarak di halaman website
5 Table <table> Tag ini digunakan untuk membuat table di halaman website
